NET PRESENT VALUE

One of the most popular project evaluation tools there is! Net Present Value calculates the difference between the investment required for any project and the cash flows of the project discounted.

INTERNAL RATE OF RETURN

Another common project evaluation tool that almost every company and investor use to decide on which projects to invest their money in - Internal Rate of Return basically calculates the breakeven rate of return on any project.

PROFITABILITY INDEX

A further analysis from the Net Present Value evaluation tool - Profitability Index is a ratio to identify the relationship between the benefits and costs of any project.

ACCOUNTING RATE
OF RETURN

A simple project evaluation tool that estimates a project's profitability and can also be called the "simple rate of return". It helps you calculate the returns of any project after deducting the costs involved.

PAYBACK PERIOD

Everything is moving so fast in today's economy - when putting money into any project, we always want to gain back our money as fast as possible. Payback Period is a tool for us to use to calculate how long it takes for us to earn our investment back in any project.
